Exhibits / Demonstrations

FlashAir Sound

Electronic musical instrument of FlashAir using sound source LSI, PSG (Programmable Sound Generator) with FlashAir.

Cleaner Robot

Desktop cleaner robot controllable with FlashAir.


TISPY

Checker of the influence of alcohol. Measure when you drink and prevent hangover. FlashAir compatible smartphone application is available.

Airio Play

Evaluation board to use FlashAir as a game controller.
You can play games using HTML and JavaScript.


Interrupt-kun

RC using FlashAir. You can carry a cup and a plastic bottle on top.

Soramame-7

Temperature/humidity data logger using FlashAir. The current value is displayed on the monitor.


Camera

Camera that can easily transfer photos to a smartphone using FlashAir.

FlashAir IoT Hub

Web service supporting self-developed device development using FlashAir. With a simple setup, you can connect FlashAir to the cloud.


GR-LYCHEE

IoT prototyping board with a camera. You can take a picture and check it with FlashAir.

Thermal printer Demo

Collaboration demo of FlashAir and a thermal printer.


FlashTools Lua Editor

Lua script editor that runs on the browser.

NFC Touch & Copy SD

Demonstration of NFC equipped SD memory card which can write data of illuminance sensor and check data with smartphone.
