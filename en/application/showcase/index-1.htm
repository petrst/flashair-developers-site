Developer: Toshiba Memory Corp.

With the FlashAir™ App for iOS and Android, you can quickly and easily download and share all the images and documents on your wireless FlashAir™ SDHC memory card.

FA Downloader

Developer: tateisu

FA Downloader scans the file list of FlashAir from Android smartphone and downloads files with specific extension. There is a one-shot mode that ends once the scan is completed, and a repetition mode to perform repeated checks and scan if there is a change.

Operating environment

  • Android 4.0 or later
  • FlashAir firmware v2.00.02 or later
  • Wi-Fi required
  • Position information embedding function is recommended since Android 4.1 or later. Play service is required.

FlashTools Lua Editor

Developer: GPS_NMEA(@Seg_faul)

FlashTools Lua Editor is a browser based script editor, which provides a code edit and easy debugging on FlashAir via Wifi.Script runs directly on the FlashAir. Don't have to worry about SD-slot corruption anymore. And, bundled IO Port tester. It could testing IO Port via browser.
Tested :Internet Explorer / Firefox / Google Chrome / Chrome for iOS / Mobile Safari

Card-Fi-Copy

Developer: Gunnar Lühr

Card-fi-copy is an application that allows you to copy pictures, videos and other files over your local network to your PC. All you need is the Toshiba FlashAir card, a WiFi network, a Windows or Mac PC and Card-Fi-Copy.

Functions at a glance:
- Automatic copying of images, videos, etc.
- Image preview for JPEG images
- Creating subfolders (such as year/month/day or source directory)
- Recognize the IP address of Wi-Fi SD Card on the network
- File filter (JPEG / MOV / misc.)

FlashAirSync

Developer: Anastasiy Safari

Main features:
- Automatically syncs remote FlashAir folder to local folder when FlashAir is present
- Eats no CPU and stays idle when there is no card present
- A single icon in menu bar
- Very simple to configure thru Settings (folders, etc.)
- Progress of sync in the drop down menu
- Overall progress in the menu bar
- Current speed and estimated time left
- Choose interactively remote FlashAir folder to sync from
- Choose interactively local folder to sync to
- Automatically scans for FlashAir and connects if present
- Uses very little memory
- Displays Mac notifications
